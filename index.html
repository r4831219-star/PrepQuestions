Ollama — 100% Local, Completely Free, No Internet Needed
⬡ Ollama runs AI models on your own machine — no API key, no account, no usage limits, fully private.
Setup (5 minutes):
  1. Download from ollama.com and install it
  2. Open terminal: ollama pull llama3.2 (3GB download, one-time)
  3. Ollama starts a local server at localhost:11434 automatically
  4. Select your model below and generate!
Recommended models: llama3.2 (3B, fast), llama3.1 (8B, better quality), mistral (7B, great for code), codellama (code-optimised)

🚀 Host on GitHub Pages — Free Forever
✅ Yes — this works 100% on GitHub Pages. It's a single HTML file with zero backend. Your AI questions are saved in browser localStorage — they survive refreshes, browser closes, and coming back days/weeks later on the same browser & device.
Step-by-step: Deploy in 3 minutes
STEP 1 — Create a GitHub repo
Go to github.com → New repository → Name it interview-prep → Public → Create
STEP 2 — Upload the file
Inside the repo → click Add file → Upload files → drag & drop interview-prep.html
⚠️ Rename it to index.html before uploading (GitHub Pages serves index.html by default)
→ Commit changes
STEP 3 — Enable GitHub Pages
Repo → Settings → scroll to Pages → Source: Deploy from branch → Branch: main → folder: / (root) → Save
✓ DONE — Your site is live at:
https://YOUR-USERNAME.github.io/interview-prep/
Takes ~1 minute to go live after saving. Share this URL with anyone!
📌 About localStorage & persistence:
• AI questions are saved in your browser's localStorage for that device/browser.
• Refresh ✅ · Close & reopen ✅ · Come back after days/weeks ✅
• Different browser or device ❌ (localStorage is per-browser)
• Incognito/private mode ❌ (cleared on window close)
• Clearing browser data/cache ❌ (manually wiped)

💡 Tip: To back up your AI questions, open browser DevTools → Console → type:
copy(localStorage.getItem('interviewPrep_aiQuestions_v1'))
Then paste it somewhere safe. To restore, run:
localStorage.setItem('interviewPrep_aiQuestions_v1', '...paste here...')
